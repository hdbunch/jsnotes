Logical Operators

  • || OR checks if at least one condition is True
  • ! NOT returns false if the result is True

if{} else{} Statement 
                if(light == "green") {
                  console.log("Drive")
              } else if (light == "orange") {
                  console.log("Get ready")
              } else if (light == "red") {
                  console.log("Dont' drive")
              } else {
                  //this block will run if no condition matches
                  console.log("The light is not green, orange, or red");
              }
Switch-Case 
switch(light) {
                case 'green':
                    console.log("Drive");
                    break;
                case 'orange':
                    console.log("Get ready");

                    break;
                case 'red':
                    console.log("Don't drive");
                    break;
                default:
                    //this block will run if no condition matches
                    console.log('The light is not green, orange, or red');
                    break;
             }

Loops perform a task over and over until a condition is met
var i = 0; is often used when making counters

for (var i =0; i<3; i++;){
console.log(i);
}
In this example the variable is set to 0, a condition of if less than 3 is set, add one more until i = 3.

while loops
                  var cnt = 0;
                  while(cnt <= 5) {
                    console.log(cnt);
                    cnt = cnt + 1;
                    }
