JavaScript Reference

Arrays are like a train with cars. In JavaScript, arrays are a type of object that allows you to store multiple values in a single variable. Creation: create an array using square brackets [ ]. For example: "var fruits = ["apple", "banana", "cherry"];"

Object literals and the dot notation
One of the most common ways of building an object in JavaScript is using the object literal syntax: var user = {}; //create an object
              //creating an object with properties and their values
var assistantManager = {
    rangeTilesPerTurn: 3,
    socialSkills: 30,
    streetSmarts: 30,
    health: 40,
    specialAbility: "young and ambitious",
    greeting: "Let's make some money"
}

Another way is with dot notaion:
              var house2 = {};
              house2.rooms = 4;
              house2.color = "pink";
              house2.priceUSD = 12345;

Both methods can be used together.
Bracket notaion: the object properties must have " " around it with bracket notation.
Properties in bracket notaion can have spaces. house2["number of rooms"] = 4 

              var house2 = {};
              house2["rooms"] = 4;
              house2['color']= "pink";
              house2["priceUSD"] = 12345;
              console.log(house2); // {rooms: 4, color: 'pink', priceUSD: 12345}

Arrays are Objects. Two of the most common built-in methods for arrays
are thepush() method and the pop() method.
push()- adds an item to the array.
 var fruits = [];
                          fruits.push("apple");
                          fruits.push('pear');
                          console.log(fruits);
                              //['apple', 'pear']

pop()- removes the last item from the array.
              
                fruits.pop();
                console.log(fruits);
                    ['apple']

A fuction can be built to create arrays:
            function arrayBuilder(one, two, three) {
              var arr = [];
              arr.push(one);
              arr.push(two);
              arr.push(three);
              console.log(arr);
            }
            arrayBuilder('apple', 'pear', 'orange')
              //   ['apple', 'pear', 'orange']

You can store the function as a variable var simpleArray = arrayBuilder('apple', 'pear', 'orange');
Now console.log(simpleArray); will log ['apple', 'pear', 'orange']

Built-in Number Constants of the Math Object
Math.PI which is approximately 3.14159
Math.E Euler's constant: which is approximately 2.718
Math.LN2 The natural logarithm of 2 which is approximately 0.693

Methods for Rounding
Math.ceil() rounds up to the next integer
Math.floor() rounds down to the next integer
Math.round() automatically rounds to the nearest integer
Math.trunc() removes the decimal numbers

Arithmetic and calculus methods
Here is a non-conclusive list of some common arithmetic and calculus methods that exist on the Math object: 

              Math.pow(2,3) - calculates the number 2 to the power of 3, the result is 8 
              
              Math.sqrt(16) - calculates the square root of 16, the result is 4 
              
              Math.cbrt(8) - finds the cube root of 8, the result is 2 
              
              Math.abs(-10) - returns the absolute value, the result is 10 
              
              Logarithmic methods: Math.log(), Math.log2(), Math.log10() 
              
              Return the minimum and maximum values of all the inputs: Math.min(9,8,7) returns 7, Math.max(9,8,7) returns 9.
              
               Trigonometric methods: Math.sin(), Math.cos(), Math.tan(), etc.


Math.random() will make a random number from 0 to 0.99. To get a larger number you can multiply it by another number.
var decimal = Math.random();
console.log(decimal);
//0.58739705
console.log(decimal * 10);
//5.8739705
