Functional programing uses a lot of functions and variables.
Data and functionality are kept seperate.
In functional programing, functions return new values which are used elswhere in the code.
Here are three important concepts in functional programing
  • First-class functions- functions are first-class citezens and can be passed around like strings or objects
  • Higher-order function- a fuction that a function as an argument, and/or a function that retruns a function
  • Pure functions and side effects- a pure function will always return the same result when given the same values, and will not make changes to anything outside of itself

In Object orented Programing (OOP)methods update properties stored in the object instead of generating new return values.
Like actual objects, the properties and data belong together in the object.
